# jaxonwallace-next

A full Next.js starter-project version of Jaxon Wallace's dark, techy link hub.

## Stack

- Next.js App Router
- TypeScript
- React components
- Tailwind v4 starter setup
- GSAP entry/hover animations

## Local dev

```bash
npm install
npm run dev
```

## Production check

```bash
npm run build
npm start
```

## Vercel

This repo is ready to import into Vercel. Use the default Vercel Next.js settings:

- Framework preset: Next.js
- Build command: `npm run build`
- Output directory: Next.js default
- Install command: `npm install`
